# How to turn on GitHub Pages (website)

**Repo:** https://github.com/Egalitarianism-EGA/ega-website  
**Public URL (after enable):** https://egalitarianism-ega.github.io/ega-website/

## Click method (easiest)

1. Open the repo: https://github.com/Egalitarianism-EGA/ega-website  
2. Click **Settings** (top menu of the repo).  
3. Left sidebar → **Pages**.  
4. Under **Build and deployment**:  
   - Source: **Deploy from a branch**  
   - Branch: **main**  
   - Folder: **/ (root)**  
5. Click **Save**.  
6. Wait 1–2 minutes.  
7. Open: https://egalitarianism-ega.github.io/ega-website/

If it 404s, wait longer or check Settings → Pages for a green check / deploy log.

## Optional

- **Custom domain:** Settings → Pages → Custom domain (you need a domain you own).  
- Edit the page: change `index.html` in the repo, commit, push — site updates automatically.

## What the site is for

Landing page for **Egalitarianism**: links to Core, mining, whitepaper, wallets.  
Not the blockchain itself — just a free homepage.
